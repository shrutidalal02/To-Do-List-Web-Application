1.To-Do List Web Application : HTML | CSS | JavaScript | Bootstrap | LocalStorage

A production-style, responsive To-Do List Web Application built to demonstrate strong fundamentals in frontend development, DOM manipulation, and client-side state management.
This project focuses on clean UI, structured code organization, and real-time task updates without any backend dependency.

2.Project Overview

The application allows users to efficiently manage daily tasks with full CRUD functionality and persistent browser storage.
It showcases practical implementation of:
Dynamic DOM rendering
Event-driven programming
Client-side data persistence
Responsive UI design

3.Core Features
Create, update, and delete tasks (Full CRUD Operations)
Real-time UI updates using JavaScript
Task completion toggle
Task filtering (All / Completed / Pending)
Persistent storage using LocalStorage API
Mobile-friendly responsive layout using Bootstrap
